# Mixtape Context
## Ubiquitous Language
**Listening Streak**: Consecutive calendar days. A streak increments when a user records a listening event on the day immediately following their last recorded listening day.
**Friends Feed Threshold**: 10 minutes. The time window within which a friend's listening activity is considered "recent" for the "Friends Listening Now" feed.
**Docstring Contract**: The docstring of a service function is the authoritative specification for its behavior. Any implementation that fails to satisfy the docstring is a bug.
